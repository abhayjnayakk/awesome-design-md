# Attribution

This repository is an independent open-source style gallery and workflow for `DESIGN.md` usage.

## Inspired By

- [Google Labs DESIGN.md](https://github.com/google-labs-code/design.md) — public format specification for describing visual identity to coding agents.
- [VoltAgent awesome-design-md](https://github.com/VoltAgent/awesome-design-md) — curated collection of DESIGN.md files inspired by popular websites.
- [getdesign.md](https://getdesign.md/) — hosted browser and request service for DESIGN.md references.

## What This Repo Adds

This repo does not mirror the full upstream collection. It adds original:

- Style taxonomy
- Workflow prompts
- Starter examples
- Selection guidance
- Remix guidance through documentation

## Brand Usage Note

References to companies or products are descriptive style references only. Their names, logos, trademarks, and visual identities belong to their respective owners.

Use references as vocabulary, not as brand cloning material.
