# DESIGN.md Style Taxonomy

Use this taxonomy to choose a visual direction before asking an agent to build UI.

## Style Families

### 1. Technical Precision

Best for:

- Developer tools
- AI infrastructure
- Security products
- Observability
- Data tools

Signals:

- High contrast
- Monospace accents
- Tight grids
- Minimal decoration
- Clear system states

Avoid:

- Decorative gradients
- Overly playful illustrations
- Marketing-first hero sections

### 2. Warm Editorial

Best for:

- AI assistants
- Writing tools
- Knowledge products
- Research apps
- Thoughtful SaaS

Signals:

- Warm neutrals
- Comfortable reading rhythm
- Serif or humanist display type
- Quiet metadata
- Calm explanatory surfaces

Avoid:

- Dense dashboards
- Cold enterprise chrome
- Excessive animation

### 3. Creator Commerce

Best for:

- Creator stores
- Marketplaces
- Checkout flows
- Digital product pages
- Publishing tools

Signals:

- Strong product cards
- Obvious prices and CTAs
- Expressive color
- Human copy
- Tangible visual surfaces

Avoid:

- Hidden pricing
- Sterile SaaS layouts
- Low-contrast pastel text

### 4. Fintech Trust

Best for:

- Payments
- Billing
- Banking
- Subscription tools
- Compliance products

Signals:

- Clear validation states
- Institutional typography
- Restrained accent color
- Accessible contrast
- Stable layouts

Avoid:

- Crypto hype
- Urgency tricks
- Decorative charts with no job

### 5. Cinematic Premium

Best for:

- Hardware
- Automotive
- Aerospace
- Luxury products
- Launch pages

Signals:

- Monumental typography
- Full-bleed imagery
- Sparse copy
- Strong black/white contrast
- Minimal UI chrome

Avoid:

- Cluttered sections
- Generic SaaS cards
- Too many accent colors

### 6. Documentation Clarity

Best for:

- API docs
- SDK products
- Dev education
- Technical knowledge bases

Signals:

- Readable side navigation
- Code blocks
- Hierarchical headings
- Search and table-of-contents patterns
- Low-friction scanning

Avoid:

- Over-designed marketing blocks
- Weak code contrast
- Hidden navigation

## Primary + Contrast Method

Choose:

- Primary family: the dominant feel.
- Contrast family: the balancing force.

Examples:

- AI devtool: Technical Precision + Warm Editorial.
- Creator checkout: Creator Commerce + Fintech Trust.
- Research SaaS: Warm Editorial + Documentation Clarity.
- Premium hardware: Cinematic Premium + Technical Precision.

## Visual Thesis Template

```text
This product should feel like [primary family] with [contrast family], so users feel [psychological outcome] while completing [main task].
```

Example:

```text
This product should feel like technical precision with warm editorial guidance, so users feel confident while configuring complex AI workflows.
```
